# 活動報告（Facebook・Instagram）用の画像

このフォルダには、`src/data/snsActivities.ts` の各投稿に紐づける画像を配置します。

## 使い方

1. 投稿内容が分かる画像ファイル（JPEG推奨）をこのフォルダに保存する。
   - ファイル名の例: `2026-07-22.jpg`（投稿日をベースにすると管理しやすい）
2. `src/data/snsActivities.ts` の該当エントリに、次のように `image` を追加する。

   ```ts
   image: {
     src: "/images/activities/2026-07-22.jpg",
     alt: "投稿内容が分かる説明文",
   },
   ```

3. `image` を省略した投稿は、オレンジ色のプレースホルダー表示のままになります。
   画像を用意でき次第、あとから追加できます。

## 画像の仕様目安

- 横長（16:9 目安）で、幅1200px程度あれば十分です（表示は自動的にカード幅・高さに合わせて
  トリミングされます）。
- Facebook・Instagramの投稿URLから画像を直接読み込む実装はしていません
  （投稿側の画像URLは期限切れや削除で表示できなくなることがあるため）。
  このフォルダに保存した画像のみを使用してください。
